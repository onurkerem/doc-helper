# doc-helper website

Marketing website for doc-helper, built with Astro and Tailwind CSS v4.

## Development

```bash
npm run dev
```

Opens at `http://localhost:4321`.

## Build

```bash
npm run build
```

Static output is written to `dist/`.

## Preview

```bash
npm run preview
```

## Design

The design system follows "The Architectural Monolith" philosophy:
- **Space Grotesk** for headlines (brutalist geometry)
- **Inter** for body text (high legibility)
- Monochromatic depth with tonal layering (no shadows/borders)
- High-contrast green accent (`#006d4a`) for interactive elements
- "Overhang" hero pattern — terminal component overlaps two sections
